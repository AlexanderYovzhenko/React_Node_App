# React_Node_App

## Download App
```
git clone https://github.com/GoldenManBel/React_Node_App.git
```

## Install npm
```
npm install
```

## Start App(server)
```
npm start
```

## Go frontend in new terminal
```
cd frontend
```

## Install npm frontend
```
npm install
```

## Start App(react)
```
npm start
```
